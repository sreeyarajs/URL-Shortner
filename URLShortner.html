Python Project: URL Shortener

This script shortens URLs using the Cutt.ly API.

Key Features:

  • Function: shorten_url - Accepts an API key and URL, returns a shortened URL or an error message.
  • Library: Uses requests for HTTP requests.
  • Error Handling: Gracefully manages HTTP and general exceptions.

How It Works:

  1. User provides a URL to shorten.
  2. API key and URL are sent to the Cutt.ly API.
  3. Response is processed and displayed to the user.

Usage:

Replace the placeholder API key with a valid one and run the script in a Python environment.
